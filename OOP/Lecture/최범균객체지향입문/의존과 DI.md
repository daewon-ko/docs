# 의존과 DI

* 의존
  * 기능 구현을 위해 다른 구성 요소를 사용하는 것
  * EX) 객체 생성, 메서드 호출, 데이터 사용
* 의존은 변경이 전파될 가능성을 의미
  * 의존대상이 바뀌면 해당 객체를 의존하는 객체 역시 바뀔 가능성이 높아짐.
  * EX) 호출하는 메서드의 파라미터가 변경, 호출하는 메서드가 발생할 수 있는 Exception Type이 추가됨
